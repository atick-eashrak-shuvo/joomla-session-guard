# Changelog

All notable changes to **Joomla Session Guard** will be documented in this file.

## 1.0.0

- Initial release.
- Joomla **4 / 5 / 6** support.
- Frontend + backend session expiry warning (modal/toast/banner).
- Manual session renewal (Extend Session) via `com_ajax` with token validation.
- Optional auto-renew before expiry.
- Cross-tab syncing (context-isolated for site vs administrator).
